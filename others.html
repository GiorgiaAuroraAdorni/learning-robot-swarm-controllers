utils.utils.extract_input_output(runs, in_label, N, communication=False, input_combination=True, myt_quantities=None)¶

Whether the input is prox_values, prox_comm or all sensors, it corresponds to the response values of ​​the sensors [array of 7 floats]. The input is normalised so that the average is around 1 or a constant (e.g. for all (dividing by 1000)). The output is the speed of the wheels (which we assume equals left and right) [array of 1 float]. There is no need to normalize the outputs. :param runs: :param in_label: :param N :param communication :param input_combination :param myt_quantities :return input_, output_, runs, columns
